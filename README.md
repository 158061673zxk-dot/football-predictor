# ⚽ 足球预测系统

智能足球预测系统，支持实时赔率获取和价值投注分析。

## 功能特点

- 🎯 智能预测算法
- 🌐 实时赔率获取
- 📊 多平台对比
- 💰 价值投注分析

## 快速开始

1. 注册 The Odds API (https://the-odds-api.com)
2. 获取免费API密钥（500次/月）
3. 部署到Vercel
4. 添加环境变量 ODDS_API_KEY

## 部署步骤

1. 上传到GitHub
2. 导入到Vercel
3. 配置环境变量
4. 自动部署完成

## 使用说明

1. 输入主队和客队
2. 点击"自动获取赔率"
3. 查看预测结果
4. 分析价值投注
